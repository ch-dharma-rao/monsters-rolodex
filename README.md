# Monsters Rolodex

## Installation

This project is built with `React`.

Clone the project from GitHub and install dependencies
```bash
git clone https://github.com/ch-dharma-rao/monsters-rolodex.git
cd monsters-rolodex
npm install
```

Then run the project with
```bash
npm start
```
